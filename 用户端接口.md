# 素材库同步


## 同步主体形象图


**接口地址**:`/api/user/v1/provider-asset-sync/sync-character`


**请求方式**:`POST`


**请求数据类型**:`application/x-www-form-urlencoded,application/json`


**响应数据类型**:`*/*`


**接口描述**:<p>根据AI模型配置智能判断是否需要上传到素材库。如果模型配置requiresAssetSync=true则上传，否则直接返回原始URL</p>



**请求示例**:


```javascript
{
  "characterId": "",
  "provider": "",
  "model": ""
}
```


**请求参数**:


| 参数名称 | 参数说明 | 请求类型    | 是否必须 | 数据类型 | schema |
| -------- | -------- | ----- | -------- | -------- | ------ |
|syncCharacterRequest|同步主体形象图请求|body|true|SyncCharacterRequest|SyncCharacterRequest|
|&emsp;&emsp;characterId|主体ID||true|string||
|&emsp;&emsp;provider|服务商 provider||true|string||
|&emsp;&emsp;model|模型名称（可选）||false|string||


**响应状态**:


| 状态码 | 说明 | schema |
| -------- | -------- | ----- | 
|200|OK|RProviderAssetSyncResponse|


**响应参数**:


| 参数名称 | 参数说明 | 类型 | schema |
| -------- | -------- | ----- |----- | 
|code||integer(int32)|integer(int32)|
|message||string||
|data||ProviderAssetSyncResponse|ProviderAssetSyncResponse|
|&emsp;&emsp;providerFileId|素材库文件ID（如果同步了）或原始URL（如果不需要同步）|string||
|&emsp;&emsp;sourceUrl|原始图片URL|string||
|&emsp;&emsp;characterId|主体ID|string||
|&emsp;&emsp;characterName|主体名称|string||
|&emsp;&emsp;synced|是否进行了同步操作|boolean||
|&emsp;&emsp;provider|服务商|string||
|timestamp||integer(int64)|integer(int64)|
|success||boolean||


**响应示例**:
```javascript
{
	"code": 0,
	"message": "",
	"data": {
		"providerFileId": "",
		"sourceUrl": "",
		"characterId": "",
		"characterName": "",
		"synced": true,
		"provider": ""
	},
	"timestamp": 0,
	"success": true
}
```
